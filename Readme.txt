## 🌟 Featured Projects

### 🧠 **EEG-Based Quadcopter Control via BCI** `Top 3 FYP`
> Real-time Brain-Computer Interface mapping EEG brainwaves directly to quadcopter flight commands with no physical controllers.
* **Stack:** Python · TensorFlow · Support Vector Machines (SVM) · Signal Processing (ICA, CSP, PSD) · EEG Hardware
* **Impact:** Trained classifier achieving 82% accuracy across 4 directional classes; awarded top 3 final-year project in the department cohort.

### 📈 **StockSense ML — AI Inventory Forecasting SaaS**
> End-to-end forecasting SaaS automating reorder alerts and warehouse demand dashboards.
* **Stack:** Python · ARIMA · Gradient Boosting · FastAPI · React.js · PostgreSQL · AWS
* **Impact:** Reduced client overstock waste by 22% and weekly forecasting effort by 90% in pilot deployment. Deployed on AWS EC2 + S3 storage.

### 💬 **LangChain RAG Customer Support Chatbot**
> Automated customer service agent containerized with Docker and deployed serverlessly.
* **Stack:** LangChain · OpenAI API · FAISS · FastAPI · Docker · AWS Lambda · MLflow
* **Impact:** Automated 60%+ of customer support queries with sub-3-second response latency. Leveraged MLflow for prompt experiment tracking.

### 🏣 **Property Intelligence Dashboard**
> Real-estate pricing heatmap and demand visualization engine.
* **Stack:** Python · Power BI · PostgreSQL · Pandas · BeautifulSoup (Web Scraping)
* **Impact:** Aggregated multi-source listings to provide price forecasting maps, saving 70% of manual research and reporting time for clients.

## 💻 Running the Portfolio Locally

You can run this project locally on your machine in under a minute.

### via Node.js
```bash
npm install -g http-server
http-server -p 8087
```
Access the site at: `http://localhost:8087`

### via Python
```bash
python -m http.server 8087
```
Access the site at: `http://localhost:8087`
