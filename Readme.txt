## 💻 Local Setup & Running

This is a static HTML/CSS/JS website. You can run it locally in your browser using any simple static file server.

### Option 1: Node.js (Recommended)
If you have Node.js installed, run a static server like `http-server`:
```bash
# Install http-server globally
npm install -g http-server

# Navigate to project directory and start server
http-server -p 8080
```
Then open [http://localhost:8080](http://localhost:8080) in your browser.

### Option 2: Python
If you have Python installed, you can start a server directly:
```bash
# For Python 3.x
python -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.
